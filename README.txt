Random Password Generator
Простое и надежное графическое приложение для генерации случайных паролей с настраиваемыми параметрами и сохранением истории.
Автор: Иван Комлач
Описание программы: Приложение позволяет пользователю быстро создавать безопасные пароли. Основные возможности:
Настройка длины: от 4 до 32 символов.
Выбор состава: цифры, буквы и спецсимволы.
История: автоматическое ведение списка ранее созданных паролей.
Автономность: сохранение данных в локальный файл history.json.
🚀 Инструкция по запуску
1. Требования
Для работы программы необходим Python версии 3.6 или выше.
2. Установка зависимостей
Программа написана с использованием стандартной библиотеки Tkinter, которая поставляется вместе с Python. Дополнительная установка сторонних библиотек через pip не требуется.
Примечание для пользователей Linux (Ubuntu/Debian): если Tkinter не установлен по умолчанию, используйте команду:sudo apt-get install python3-tk3. 
Запуск программы
Перейдите в папку с файлом Random Password Generator.py и выполните в терминале или командной строке следующую команду: python "Random Password Generator.py"
